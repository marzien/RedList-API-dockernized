## Quick Start

```bash
# Install dependencies
npm install

# Run the React client
npm start

# Client runs on http://localhost:3000
```

##

Project should have globar variable file `.env` with content:

`REACT_APP_API_KEY=<YOUR_KEY_GOES_THERE>`

## Docker image

Fire up container:

```
docker-compose up
```

## Testing

Run all tests:

```
npm test
```
